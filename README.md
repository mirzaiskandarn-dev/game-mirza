# STICKMAN DRIVE: PROTOCOL

Sebuah game fighting stickman bergaya cyberpunk yang dibuat dengan React, Vite, dan Tailwind CSS.

## Fitur
- **Dynamic Physics**: Pergerakan stickman yang lentur dan responsif.
- **Unique Characters**: Karakter dengan aksesori dan efek spesial (Inferno, Volt, Shadow, Pulse, Blade).
- **Ultimate Abilities**: Serangan hebat yang bisa digunakan setelah mengumpulkan energi.
- **Cross-Platform**: Bisa dimainkan di PC (Keyboard) dan Mobile (Joystick).
- **Hardened Security**: Integrasi Firebase siap pakai untuk leaderboard (opsional).

## Cara Menjalankan
1. Instal dependensi:
   ```bash
   npm install
   ```
2. Jalankan mode pengembangan:
   ```bash
   npm run dev
   ```
3. Bangun untuk produksi:
   ```bash
   npm run build
   ```

## Kontrol Keyboard
- **WASD**: Gerak & Lompat
- **J**: Serangan Utama
- **K**: Bertahan (Block)
- **L**: Ultimate (Butuh 50% Energi)

## Teknologi
- React 19
- Framer Motion (untuk animasi UI)
- Lucide React (untuk ikon)
- HTML5 Canvas (untuk gameplay)
- Express (sebagai server backend)
